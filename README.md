# 学習ログアプリ

Next.js と Supabase を使用して開発した、シンプルな学習記録アプリケーションです。

## 💡 デプロイURL
[アプリにアクセスする](https://my-portfolio-app-psi-pied.vercel.app/logs)
※テスト用アカウント：user@example.com / パスワード: test

## 🚀 開発背景
プログラミング学習や資格試験の勉強において、「前回どこまで進めたか忘れてしまう」「学習の継続実感が得られにくい」という自分自身の課題を解決するために開発しました。
学習のモチベーションを維持するため、**「迷わず、直感的に、毎日の進捗を10秒で記録できること」**を意識したUI/UXを目指しました。

## ✨ 主要機能
- **ユーザー認証機能**（Supabase Authによる安全なサインイン/サインアウト）
- **学習ログのCRUD機能**（毎日の学習内容の登録・一覧表示・削除）
- **レスポンシブ対応**（スマートフォンからでも手軽に記録可能）

## 🛠️ 使用技術
- **フロントエンド:** Next.js (App Router), Tailwind CSS, shadcn/ui
- **バックエンド/データベース:** Supabase (PostgreSQL, Supabase Auth)
- **ホスティング:** Vercel

## 🔥 苦労した点・学んだこと（技術的チャレンジ）
JavaScriptおよびReactの学習からスタートしたため、最初は状態管理（State）やコンポーネントのレンダリングのライフサイクルを理解することに苦戦しました。
特に、Supabaseからデータを取得する際の非同期処理の挙動において予期せぬエラーが発生した際は、単に生成AIのコードをコピペするのではなく、**公式ドキュメントを参照し、開発者ツールやデバッグを用いてデータの流れを1行ずつ追う**ことで原因を特定しました。
この経験を通じ、モダンなフロントエンド開発の基礎となる「データフローの意識」と、自力でエラーを解決する「デバッグ力」を養うことができました。

## 開発環境の起動方法

ローカル環境で起動する場合は、以下のコマンドを実行してください。

```bash
npm install
npm run dev
```
(URL: http://localhost:3000)
